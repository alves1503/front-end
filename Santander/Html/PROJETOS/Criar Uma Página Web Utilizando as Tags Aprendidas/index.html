Trabalhando com tabelas em HTML

Conteúdos aprendidos:

  • Sobre Tabelas
  • Tag table
  • Tag tr
  • Tag td e th
  • Tag tbody, thead e tfoot
  • Estilizando Tabelas

A tag "table" é utilizada no HTML para criar tabelas, que organizam dados em linhas e colunas, facilitando a visualização de informações estruturadas, como listas, cronogramas ou relatórios.
Principais tags de uma tabela:
"table": define o início e o fim da tabela.
"tr" (table row): define uma linha da tabela.
"th" (table header): define um cabeçalho de coluna, geralmente exibido em negrito e centralizado.
"td" (table data): define uma célula comum da tabela, onde os dados são inseridos.

A semântica no HTML refere-se ao uso de tags que têm significado claro e descritivo, indicando ao navegador (e aos desenvolvedores) qual é o papel daquele conteúdo na página. Isso melhora a acessibilidade, organização, manutenção do código e também o SEO (otimização para buscadores).

Tag Função
<header> Cabeçalho da página ou de uma seção.
<nav> Contém links de navegação.
<main> Representa o conteúdo principal da página.
<section> Define uma seção temática do conteúdo.
<article> Conteúdo independente (ex: post de blog, notícia).
<aside> Conteúdo secundário (ex: barra lateral, anúncios).
<footer> Rodapé da página ou de uma seção.
<figure> Agrupa imagens e suas legendas.
<figcaption> Define a legenda de uma figura.
<mark> Destaca um trecho do texto.
<time> Representa data ou hora.

Alem disso o SEO (Otimização para Motores de Busca) é o conjunto de estratégias e técnicas usadas para fazer com que um site apareça nas primeiras posições dos resultados de buscadores como o Google, sem pagar por anúncios.
